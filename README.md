
# Ekon-Sons SEO Website Starter

This is a simple SEO-focused Next.js website starter for Ekon-Sons LTD.

## Features
- SEO optimized
- Location + service pages
- WhatsApp CTA
- Vercel-ready deployment
- Blog/article support
- Simple scalable structure

## Install

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this project to GitHub
2. Go to https://vercel.com
3. Import GitHub repository
4. Deploy

## Important SEO Strategy

Create many pages like:
- /roof-leak-repair-east-legon
- /waterproofing-tema
- /emergency-plumber-spintex

Each page should target:
- one service
- one location
- one problem
